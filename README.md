# Calculator-Projekt
# Mini Calculator (Python)

## Overview

This project is a simple command-line calculator written in Python. The program allows the user to enter two numbers and perform one of four basic arithmetic operations: addition, subtraction, multiplication, or division.

The purpose of this project is to demonstrate basic Python concepts such as user input, conditional statements, and simple mathematical operations.

---

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Basic error handling for division by zero

---

## How the Program Works

1. The program starts by displaying a message indicating that the calculator has started.

2. The user is asked to enter the first number.

3. The user selects the operation they want to perform (`+`, `-`, `*`, or `/`).

4. The user enters the second number.

5. The program checks which operation was selected using conditional statements (`if`, `elif`, `else`).

6. The selected calculation is performed.

7. The result is printed to the terminal.

---

## Example

Example interaction in the terminal:

```
Mini Calculator
Enter first number: 10
Choose operation (+, -, *, /): *
Enter second number: 5
Result: 50
```

## Technologies Used

## How to Run the Program

1. Clone the repository or download the files.
2. Navigate to the project folder in your terminal.
3. Run the following command:

```
python calculator.py
```

or

```
python3 calculator.py
```
